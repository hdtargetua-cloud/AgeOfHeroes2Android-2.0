# Age of Heroes 2 — Android Port

Порт оригинальной J2ME игры (Qplaze/mikhan, 2006) на платформу Android 10+.

## Что сделано

### Управление
- **Tap-to-move**: тапни на любой тайл — герой автоматически найдёт путь (BFS)
- **Свайп**: перетаскивание карты с инерцией
- **Долгое нажатие**: информация об объекте
- **HUD кнопки**: "Конец хода", "Сохранить" — крупные, удобные для пальца

### Графика
- Оригинальные PNG спрайты масштабированы x3 (240×320 → 720×960+)
- Nearest-neighbor масштабирование — сохранён пиксель-арт стиль
- Туман войны
- Плавная анимация движения героя

### Расширенный контент (для большого экрана)
- Карта расширена с ~18×22 до 26×32 тайлов
- 8 новых нейтральных армий для вербовки
- 8 новых врагов (мобы на карте)
- Дополнительные шахты, сундуки, артефакты

### Фикс бага зданий (бесконечный фарм)
- **Проблема оригинала**: повторный вход в здание давал армию снова и снова
- **Решение**: каждое здание можно посетить 1 раз за неделю
- Ключ хранения: `x:y:week` — автоматически устаревает при смене недели
- Сброс только при реальной смене игровой недели

### Сохранение
- Автосохранение при паузе (уход из игры)
- Ручное сохранение кнопкой в HUD
- Сохраняется: золото, армия, карта, туман войны, посещённые здания

## Сборка

### Требования
- Android Studio Hedgehog или новее
- Android SDK 29+
- Java 11

### Шаги
1. Открыть папку `AgeOfHeroes2Android` в Android Studio
2. В `local.properties` указать путь к SDK:
   ```
   sdk.dir=/Users/username/Library/Android/sdk
   ```
3. Build → Make Project
4. Run on device/emulator (Android 10+)

## Структура проекта

```
app/src/main/
├── java/com/aoh2/
│   ├── MainActivity.java       # Главное меню
│   ├── GameActivity.java       # Игровой экран + диалоги
│   ├── data/
│   │   ├── GameConstants.java  # Все константы
│   │   ├── GameState.java      # Состояние игры + фикс бага
│   │   └── MapData.java        # Данные карты + новый контент
│   ├── engine/
│   │   ├── SpriteManager.java  # Загрузка спрайтов x3
│   │   ├── MapRenderer.java    # Рендеринг карты
│   │   └── TouchHandler.java   # Тач управление
│   ├── game/
│   │   ├── BattleEngine.java   # Пошаговый бой
│   │   └── HeroController.java # Движение героя + pathfinding
│   └── ui/
│       └── GameView.java       # Главный Canvas + game loop
└── assets/
    ├── sprites/    # 0.png..32.png — оригинальные спрайты
    ├── data/       # r.bin, t.bin, q.bin — данные карты
    └── music/      # 0.mid..6.mid — музыка
```
